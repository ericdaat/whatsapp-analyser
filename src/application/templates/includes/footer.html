About

"Whatsapp Analyzer" is a data visualisation tool that lets you explore group conversations history. It show insights about who posted the most, when, what kind of words they used and what are the main topics of the conversation. It's built in Python using mainly Flask and Plotly libraries.
The code is open source and available on my Github.
